# Analyzing E-Learning Platform Purchases Using MySQL

## Project Overview

This project analyzes purchase data from an online learning platform using MySQL. The objective is to understand learner behavior, sales performance, course popularity, and category-wise revenue through SQL-based data analysis.

## Database Structure

The project consists of three relational tables:

* **learners** – Stores learner information.
* **courses** – Stores course details, categories, and prices.
* **purchases** – Stores purchase transactions and links learners with courses.

## SQL Concepts Used

* Database Creation and Table Design
* Primary Keys and Foreign Keys
* Data Insertion (DML)
* INNER JOIN, LEFT JOIN, RIGHT JOIN
* Aggregate Functions (SUM, AVG, COUNT)
* GROUP BY and HAVING
* Subqueries and Correlated Subqueries
* Common Table Expressions (CTE)
* CASE Expressions
* NULL Handling using IFNULL/COALESCE
* Views
* Business-Oriented Analytical Queries

## Key Insights

* Data Science courses generated the highest revenue.
* Beginner-level courses attracted the largest number of learners.
* High-value learners spent more than ₹15,000 on course purchases.
* Multiple learners enrolled across different course categories.
* Revenue and purchase trends were analyzed to support business decisions.
